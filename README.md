# xmail
Golang tool that detects which email addresses have domains which are able to be registered

## Install:

```
go install github.com/haccer/xmail@latest
```

or

```
git clone https://github.com/haccer/xmail
cd xmail
go install
```

## Usage:

```
cat emails.txt | xmail
```
